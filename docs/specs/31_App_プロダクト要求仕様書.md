# Appプロダクト要求仕様書（PRD）

## 1. Appの使命
NightShiftOS Appは、スタッフの最小操作で、入退室記録・危険通知・ケア記録を確実に実現する。

## 2. 要求定義
| ID | 要求 | 優先度 | 受入条件 |
|---|---|---|---|
| AR-001 | バックグラウンド入退室検知 | Must | Beacon検知でENTER/EXIT自動送信 |
| AR-002 | 危険通知を確実に表示 | Must | マナーモード時も高優先通知表示 |
| AR-003 | オフライン時の記録保全 | Must | 圏外でも記録欠落なし |
| AR-004 | 手動ケア記録を1タップで入力 | Must | 主要ケア種別を即時記録 |
| AR-005 | 夜勤向け低認知負荷UI | Should | ダークテーマ・大ボタン |
| AR-006 | バッテリー影響を最小化 | Must | 1時間あたり消費2%以下目標 |

## 3. 非機能
- 起動から業務開始まで <= 15秒
- 画面未操作時のバックグラウンド安定稼働
- ローカルデータ暗号化

## 4. 対応OS
- iOS 16+
- Android 12+
