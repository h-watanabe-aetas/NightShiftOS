# Edgeプロダクト要求仕様書（PRD）

## 1. Edgeの使命
NightShiftOS Edgeは、居室内の状況をプライバシー侵害なく推定し、Beaconによる位置文脈を常時提供する。

## 2. 要求定義
| ID | 要求 | 優先度 | 受入条件 |
|---|---|---|---|
| ER-001 | ミリ波レーダーから在床状態を推定 | Must | `SLEEP/SITTING/OUT_OF_BED` の3状態を出力 |
| ER-002 | iBeaconを常時送信 | Must | 電源投入5秒以内に広告開始 |
| ER-003 | Wi-Fi断でも状態推定継続 | Must | 推定停止時間0、再接続後再送 |
| ER-004 | OTAで安全に更新 | Must | 失敗時ロールバック可能 |
| ER-005 | 機種差を吸収する抽象化 | Should | Radar Driverを差替可能 |
| ER-006 | 夜間環境に配慮した表示 | Should | LED輝度上限10% |

## 3. 非機能
- 稼働率: 99.5%（端末単体）
- 起動時間: 10秒以内
- 温度: ケース内70℃未満
- 復旧: ハング時5秒以内自動再起動

## 4. 運用前提
- 電源常時供給（5V/1A以上）
- 設置高1.0〜1.5m、対象ベッド方向へ固定
- 初期導入時に施設ごとの閾値校正を実施
